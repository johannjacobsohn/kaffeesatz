Kaffeesatz(media)

This is just another toy project, comprising a server, an admin interface and an user app.

            Server

User App            Admin Interface

Server

The server (in /server) serves both the admin interface and the user app (both are static). Its written in NodeJS using the express framework and offers a REST-Interface and accepts websocket connections (thanks to socket.io).

Data is save to and retrieved from a mongoDB (via mongojs).

At the moment the server does not enforce any type of security or data validation - in fact it does get corrupted easily. Its covered by mocha tests (see /server/tests) and is hosted at appfog.

I has its own readme file at /server/readme.md.

Admin Interface

The administration is written with backbone.js and Twitter Bootstrap.

I has its own readme file at /admin/readme.md.

User App

The user app is an enyo-app using the very new and exciting mvc-plugin. It's directly derived from the Bootplate-MVC-App.

I has its own readme file at /app/readme.md.

Goals

This project aims to:

  • have a good test coverage
  • have some server/client code sharing
  • nice interface
  • good documentation
  • to feature on screen guidance for both app and admin section
  • be installable on ios & firefox
